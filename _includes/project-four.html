Pocket Fit

I had to build a full-stack app using Ruby on Rails and AngularJS. I decided to create a gym app called “Pocket Fit”. This app was dedicated to fitness. First of all, I planned my week using Trello to arrange my schedule and deadlines. After completing the pseudocoding, I used Justinmind wireframing software to design the user interface and layout. I then started creating the database models to build the API.

Once I had completed this, I started building my UI, which comprised a cool way to select and filter exercises by combining group muscles and intensity level. For the first time, I used Adobe Photoshop which allowed me to create a muscle group selector. Finally, I used a Tachyons framework to create the styling, and Slick to create carousels.
