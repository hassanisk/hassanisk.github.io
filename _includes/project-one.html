Alien Attack

I had a week to a build an interactive game using HTML, CSS, SCSS, JavaScript and jQuery. I first had to come up with an idea for the project. After a discussion with the Teaching Assistant, I decided to create a ‘duck hunt’ style game called Alien Attack.

I started with planning and pseudocoding the process. I then began developing the functionality and building the logic of the game. After that, I then started building the CSS and giving the game a ‘cool’ and ‘modern’ look.

This project was extremely insightful for me, as it was the first project that I created on my own.
