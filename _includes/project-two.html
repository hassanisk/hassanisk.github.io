MoviePedia

I had to build a full-stack app using Node.js and Express. I created a movie search app called MoviePedia.

For this project, I had to create an app including at least two models with user authentication. I used an external API for movies called OMDB. Following the first project process, I used Balsamiq software to create the wireframes. I then developed the user interface and navigation. In this project, I used the Bootstrap framework.
