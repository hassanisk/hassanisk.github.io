Pocket Fit

I had to build my first full-stack app using Ruby on Rails and AngularJs. I created a gym app called “Pocket Fit”.
For this last project, I decided to create an app dedicated to fitness. First of all, I planned my week using Trello to arrange my schedule and deadlines. Then after that, the pseudocoding, I used Justinmind wireframing software to design the user interface and layouts. Then, I started creating the database models to build the API.

Once done, I started building my UI, a cool way to select and filter exercises combining group muscles and intensity level. For the first time ever, I used Adobe Photoshop to help me create a great muscles groups selector, a new skill I am happy to take. In addition, I used Tachyons framework to create the styling, and Slick to create some slick carousels.
